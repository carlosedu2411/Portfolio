Descrição do Projeto

Vitalmen é um site dedicado ao autocuidado e bem-estar masculino. O projeto foi desenvolvido com uma arquitetura completa incluindo frontend responsivo e backend robusto, demonstrando conhecimento em desenvolvimento full-stack.

Tecnologias Utilizadas

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python com Flask
  • Banco de Dados: PostgreSQL
  • Arquitetura: Model-View-Controller (MVC)

Funcionalidades Principais

  • Interface intuitiva e responsiva
  • Sistema de autenticação de usuários
  • Banco de dados relacional para gerenciar informações
  • Rotas dinamicamente processadas pelo Flask
  • Validação e tratamento de dados no backend

Aprendizados

  • Integração entre frontend e backend
  • Design de banco de dados relacional
  • Desenvolvimento de APIs RESTful com Flask
  • Boas práticas em segurança web
