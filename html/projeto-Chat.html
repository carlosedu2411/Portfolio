Descrição do Projeto

O DevChat é uma aplicação de mensagens instantâneas moderna e responsiva. Desenvolvido com foco em alta performance e escalabilidade, o sistema conta com um ecossistema robusto de backend que gerencia conexões persistentes, permitindo a troca de mensagens em tempo real com armazenamento seguro e relatórios eficientes de persistência de dados.

Arquitetura e Tecnologias

  • Backend API: Node.js estruturado de forma escalável e performática.
  • Banco de Dados: PostgreSQL gerenciando de forma robusta o histórico e dados de usuários.
  • ORM: Prisma ORM intermediando a comunicação com o banco através de queries otimizadas.
  • Cloud/Cloud Hosting: API Backend e Banco de Dados 100% hospedados e implantados na nuvem do Railway.
  • Frontend Interface: Layout moderno com tema escuro (Dark Mode) focado na experiência do usuário e adaptabilidade mobile.

Funcionalidades Chave

  • Envio e recebimento de mensagens com atualização instantânea
  • Painel lateral expansível para busca e seleção de usuários
  • Indicador de status online/offline para os contatos
  • Interface fluida simulando o design de plataformas modernas de comunicação
  • Persistência integral do histórico de conversas direto no banco de dados

Deploy e Infraestrutura

  • Railway CI/CD: Deploy automatizado integrado diretamente ao repositório.
  • Segurança: Variáveis de ambiente protegidas na nuvem para conexão com a URI do Postgres.
  • Prisma Migrations: Controle e versionamento do schema do banco diretamente em produção.
