Descrição do Projeto

Uma aplicação web interativa que permite aos usuários criar, visualizar e gerenciar comentários. Os dados são armazenados em um banco de dados local, demonstrando conhecimento em persistência de dados e manipulação do DOM com JavaScript.

Tecnologias Utilizadas

  • Frontend: HTML5, CSS3, JavaScript Vanilla
  • Armazenamento: LocalStorage / IndexedDB
  • Arquitetura: Single Page Application (SPA)

Funcionalidades Principais

  • Adicionar novos comentários em tempo real
  • Listar todos os comentários salvos
  • Persistência de dados no navegador
  • Interface intuitiva e responsiva
  • Validação de entrada de dados
  • Possibilidade de deletar comentários

Aprendizados

  • Manipulação do DOM com JavaScript puro
  • Uso de LocalStorage para persistência de dados
  • Eventos do JavaScript (click, input, etc)
  • Estruturação de dados em JSON
  • Design responsivo para diferentes dispositivos
